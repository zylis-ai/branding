# Brand Colors

This directory contains the official Zylis color palette.

## Palette

| Color Name | Hex Code | Description |
| :--- | :--- | :--- |
| **Zylis Cyan** | `#00CAD6` | Primary brand color. |
| **Dark Charcoal** | `#333333` | Primary text and background color. |
| **White** | `#FFFFFF` | Secondary background and text color. |

For programmatic usage, refer to [`palette.json`](palette.json).
