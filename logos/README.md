# Logos

This directory contains official Zylis logo assets organized by use case.

## Directory Structure

- **[vector](vector/)**: SVG files for print and high-resolution digital use. Best for scalability.
- **[web](web/)**: Optimized files for web applications.
- **[social](social/)**: Assets formatted for social media profiles and posts.
- **[favicon](favicon/)**: App icons and browser favicons.

## Guidelines

- Do not alter the aspect ratio of the logos.
- Ensure sufficient contrast with the background.
- Use the provided dark mode assets when placing the logo on dark backgrounds.
