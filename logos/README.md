# Logos

This directory contains official Zylis logo assets organized by use case.

## Directory Structure

- **[vector](vector/)**: SVG files for print and high-resolution digital use. Includes Affinity Designer source files.
- **[monochrome](monochrome/)**: Black and white versions of the logo.
- **[favicon](favicon/)**: Web icons, favicons, and social sharing metadata images.
- **Root Files**:
  - `logo-full.png`: Primary full-color logo.
  - `logo-full-dark-mode.png`: Full-color logo optimized for dark backgrounds.
  - `logo-icon.png`: Standalone brand icon.
  - `social-profile.png`: Optimized for social media profile pictures.

## Guidelines

- Do not alter the aspect ratio of the logos.
- Ensure sufficient contrast with the background.
- Use the provided dark mode assets when placing the logo on dark backgrounds.
