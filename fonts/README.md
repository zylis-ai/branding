# Typography

The official typeface for Zylis is **Figtree**.

## Font Families

- **Figtree Regular**: Used for body text and general content.
- **Figtree ExtraBold**: Used for headlines and emphasis.

- [`Figtree-Regular.ttf`](Figtree-Regular.ttf)
- [`Figtree-ExtraBold.ttf`](Figtree-ExtraBold.ttf)
